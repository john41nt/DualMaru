# DualMaru
A project to record PC games support status of press 〇 (Circle, or "Maru" in Japanese) button to confirm with DualShock 4 and DualSense controllers. Currently, most PC games default to the Xbox controller layout, where the A button serves as 'Confirm'. On a PlayStation controller, this action is mapped to the X (Cross, or "Batsu" in Japanese) button. The goal of this project is to document and catalog which games have been properly adapted for O button confirmation.

这是一个用于汇总 PC 游戏是否支持连接索尼 DualShock 4 及 DualSense 控制器时使用 〇（圈，即日语的まる）键进行确认动作的项目。目前大多数 PC 游戏都默认使用 Xbox 面板的 A 键，即 PlayStation 控制器对应位置的 X（叉，即日语的ばつ）键输入确认动作。本项目的目的是记录和整理 PC 游戏支持 〇 键确认与否。

このプロジェクトは、PC ゲームが SONY DualShock 4 または DualSense コントローラー使用時、〇（まる）ボタンによる「決定（確認）操作」をサポートしているかを一覧・まとめするものです。現在、多くの PC ゲームは Xbox コントローラーの A ボタンまたは PlayStation コントローラーでの対応位置 X（ばつ）ボタンを決定操作のデフォルトとして採用しています。本プロジェクトでは、各タイトルにおける「 〇 ボタン確定」の有無​を記録・整理し、情報を共有することを目的としています。
